# steps
follow the instructions on Tensorflow.org (TF) for "Install TensorFlow with pip":
to create the virtual env (venv)
```
python3 -m venv --system-site-packages ./venv
```

to start the venv
```
source ./venv/bin/activate  # sh, bash, or zsh
```

to stop:
```
deactivate
```
